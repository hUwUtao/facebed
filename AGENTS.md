# AGENTS.md

This repo is Java now. Treat it like a Spring Boot app, even when it is doing foolish things.

Rules:

- Prefer small, direct changes.
- Preserve the existing routes and assets.
- Keep the Spring bean cosplay intact.
- If you touch docs, keep the ranty tone.
- Mention shared authorship if you are writing notes or release text.
- Verify with a clean Maven build when possible.

Useful commands:

```bash
mvn clean compile
mvn -DskipTests package
```
